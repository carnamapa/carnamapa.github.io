Sobre o Projeto CarnaMapa

Um projeto para nos encontrarmos no Carnaval de rua de São Paulo

Criamos esse projeto simplesmente por não ter encontrado as informações dos blocos de rua de São Paulo geolocalizadas. Em um dia comum de carnaval, acompanhando um bloquinho já nos seus finalmentes, quem nunca quis encontrar um outro bloquinho para continuar a festa?

Os dados foram extraídos através de um script em python que faz o scrapper das informações dos blocos de rua do site Blocos de Rua. Essas mesmas informações estão disponíveis no site da prefeitura de São Paulo.
